# Locations — Raum-Namen + Aliase

Snipe-IT hat feste Location-Namen. Benutzer sagen oft Alltags-Namen.
Hier mappen wir beides.

Format pro Zeile:
`Snipe-IT-Name — Aliase: <wörter> — Hinweis: <optional>`

## Regeln

- Benutzt der User einen Alias, den du hier findest: **den
  Snipe-IT-Namen verwenden**, den Alias im Vorschlag einmal kurz
  erwähnen (*"Werkstatt (Workshop-1)"*).
- Ist der Alias unklar oder mehrdeutig: fragen, nicht raten.
- Fehlt der Ort in Snipe-IT komplett (`manage_locations` list hat
  ihn nicht): Benutzer fragen, ob neu anlegen — nicht stumm
  erstellen.
